# LatihanSoalUKL2No2
Fungsi Utama:Program ini berfungsi sebagai kalkulator untuk menghitung volume tabung menggunakan rumusnya
Cara Kerja: Program meminta dua masukan dari user: jari-jari (r) dan tinggi (t),Angka-angka dikirim ke penghitungan (hitungVolume),Fungsi tersebut kemudian menghitung volume menggunakan rumus: 3.14 (yang disederhanakan) dikali jari-jari kuadrat dikali tinggi,dan hasil volume tabung tersebut lalu ditampilkan kepada user.
